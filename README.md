# ufc.stats

<!-- badges: start -->

<!-- badges: end -->

ufc.stats is a data package containing historical data from UFC events.
The data dates back to the very first fight in 1993 and is continously
updated with data from new UFC events.

## Installation

You can install the development version from
[GitHub](https://github.com/) with:

``` r
# install.packages("devtools")
devtools::install_github("mtoto/ufc.stats")
```
